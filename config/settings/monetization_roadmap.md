## 1. 수익화 시작 가능 시점

### 팔로워별 수익화 가능 범위

1,000명: 제휴 마케팅 시작 가능 (쿠팡파트너스 등, 팔로워 제한 없음)
3,000명: 제품 협찬 시작 (제품만 제공, 원고료 없음)
5,000명: 소액 원고료 + 제품 협찬 (5~30만원)
10,000명: 본격 협찬 + 인스타그램 공식 수익화 도구 사용 가능
30,000명: 브랜드 앰배서더 계약 가능
50,000명: 캠페인급 협업 (150~300만원)
100,000명: 브랜드 파트너십 (300만원+)

### 결론
쿠팡파트너스는 즉시 시작 가능 (팔로워 제한 없음)
제품 협찬은 3,000명부터 현실적
유의미한 수익은 10,000명부터

## 5. 단계별 수익화 로드맵

### Phase 1: 씨앗 단계 (0~3,000명 / 1~2개월차)

목표: 제휴 마케팅 기반 구축
수익 예상: 월 0~5만원

실행:
- 쿠팡파트너스 가입 및 링크 생성
- 캡션에 "추천 간식 링크는 프로필 링크트리" 삽입
- 링크트리 또는 비핸들 세팅 (제휴 링크 모음)
- 콘텐츠에 자연스럽게 제품 노출 (사과깎이, 간식통 등)

주의:
- 광고 표시 의무 준수 (#광고 #협찬 #제휴링크)
- 과도한 광고 느낌 금지 (정보 90% + 광고 10%)

### Phase 2: 발아 단계 (3,000~10,000명 / 3~4개월차)

목표: 제품 협찬 시작
수익 예상: 월 5~30만원 (제품 현물 포함)

실행:
- 링크튜브/블라이지 가입, 체험단 신청
- 미디어킷 제작 (계정 소개, 참여율, 콘텐츠 예시)
- 소형 브랜드(바잇미, 보우와우)에 DM 협업 제안
- 월 1~2건 협찬 콘텐츠 진행
- 스토리 하이라이트에 "리뷰" 카테고리 추가

콘텐츠 비율: 정보 80% + 협찬리뷰 15% + 광고 5%

### Phase 3: 성장 단계 (10,000~30,000명 / 4~5개월차)

목표: 정기 수익 발생
수익 예상: 월 30~100만원

실행:
- 인스타그램 크리에이터 계정 전환 (릴스 보너스 활성화)
- 중형 브랜드(하림펫푸드, 내추럴코어)에 접근
- 월 3~5건 유료 협찬 (30~70만원/건)
- 제휴 마케팅 수익 최적화 (고수익 제품 선별)
- 릴스 콘텐츠 추가 (조회수 수익화)

### Phase 4: 수확 단계 (30,000명+ / 6개월차~)

목표: 안정적 수익 구조
수익 예상: 월 100~300만원+

실행:
- 대형 브랜드(로얄캐닌, 오리젠) 앰배서더 계약
- 월 5~8건 유료 협찬 (70~150만원/건)
- 자체 굿즈/가이드북 판매 (햇살이 캐릭터 활용)
- 온라인 강의/전자책 ("강아지 영양 가이드")
- 라이브 방송 수익화

## 9. @sunshinedogfood 수익화 장점

### 경쟁 우위
- 교육 정보 계정 → 신뢰도 높음 → 협찬 전환율 높음
- 저장율 높은 콘텐츠 → 장기 노출 → 제휴링크 클릭 지속
- 자동화 시스템 → 콘텐츠 생산 비용 최소 → 높은 마진
- 팩트체크 기반 → 수의사/영양사 협업 가능

### 차별화된 수익 모델
- 일반 펫스타그램: 귀여운 사진 → 단순 협찬
- @sunshinedogfood: 영양 정보 교육 → 사료/간식 리뷰 신뢰도 극대화 → 높은 전환율
