## 4. 스토리 흐름

```
[시작] 표지: 호기심 유발 "이거 먹어도 될까?"
    ↓
[전개] 01~04: 정보 전달 (안전성, 영양소, 효능)
    ↓
[전환] 05~06: 주의사항과 적정량
    ↓
[마무리] 07: CTA "저장해두세요!"
```

### 감성 흐름
```
호기심 → 안심 → 기대감 → 조심 → 행동 유도
```
