## 핵심 발견

### 1. 시장 공백 (기회)
```
한국어 + 강아지 음식 영양 정보 + 인포그래픽 캐러셀 = 거의 없음!
```

- **영어권**: 강아지 영양 전문 계정 다수 (수의사, 영양사 기반)
- **한국어권**: 귀여움 콘텐츠 or 브랜드 홍보 위주
- **공백**: 한국어로 된 체계적 강아지 음식 정보 교육 계정

### 2. 우리의 차별점

| 경쟁 계정 | 우리 (@sunshinedogfood) |
|-----------|------------------------|
| 텍스트 위주 or 사진 1장 | AI 생성 고품질 캐러셀 10장 |
| 수동 제작 (주 1~2회) | 자동화 (하루 3회) |
| 전문 용어 | 쉬운 한국어 + 이모지 |
| 사람 중심 | 햇살이(골든리트리버) 캐릭터 |
| 단편적 정보 | 효능+주의사항+급여법 체계적 |

### 3. 벤치마크 전략

#### 콘텐츠 형식 (캐러셀 10장 구조)
```
S0: 표지 ("강아지 OO 먹어도 되나요?")

## 콘텐츠 전략 제안

### 주간 콘텐츠 믹스 (하루 3개)
```
월: 안전 과일 + 안전 채소 + 위험 식품 경고
화: 안전 육류 + 안전 해산물 + 급여 팁
수: 안전 과일 + 안전 곡물 + 위험 식품 경고
목: 안전 채소 + 보충제 + 급여 팁
금: 안전 육류 + 안전 과일 + 위험 식품 경고
토: 주간 베스트 리포스트 + 신규 2개
일: Q&A 정리 + 신규 2개

### 참여율 높은 콘텐츠 유형 (벤치마크 기반)
1. **"OO 먹여도 되나요?" 시리즈** - 검색 의도 일치
2. **"절대 먹이면 안되는 TOP 5"** - 공유/저장 폭발
3. **"수의사가 추천하는 간식"** - 신뢰도
4. **"계절별 추천 간식"** - 시의성
5. **"연령별 급여 가이드"** - 저장율 높음
