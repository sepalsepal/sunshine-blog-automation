# 🤖 AI Crew 사용 설명서

## 개요

Gemini API 기반 AI Crew 시스템입니다.  
기존 CrewAI 대비 **무료 + 빠름 + 간단한 구조**입니다.

---

## 에이전트 구성

| 페르소나 | 역할 | 엔진 |
|----------|------|------|
| 김부장 | 전략/총괄 | Claude (별도) |
| 최과장 | 코딩/실행 | Claude Code (별도) |
| **김차장** | 콘텐츠 기획 | Gemini 1.5 Flash |
| **김작가** | 텍스트 생성 | Gemini 1.5 Flash |
| **최검증** | 팩트체크 | Gemini 1.5 Flash |
| **박과장** | 품질 검수 | Gemini 1.5 Flash |
| **이카피** | 캡션 작성 | Gemini 1.5 Flash |

---

## 설치

```bash
pip install google-generativeai
```

---

## 사용법

```bash
# 기본 실행
python ai_crew.py peach

# 다른 주제
python ai_crew.py watermelon
python ai_crew.py broccoli
```
