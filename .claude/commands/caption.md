# /caption - 캡션 작성

## 필수 인자
- `$ARGUMENTS` = 콘텐츠 번호 또는 이름 (예: 060, 후라이드치킨)

## 인자 검증
인자가 없으면 아래 메시지 출력 후 중단:
```
사용법: /caption <콘텐츠번호 또는 이름>
예시: /caption 060
      /caption 후라이드치킨
```

## 실행 절차

### 1. RULES.md 확인
- §6.2 캡션 규칙 (파스타 규칙)
- §6.3 보호자 동질감 규칙
- §6.5 포스트 제목 규칙
- §6.6 급여량 작성 형식

### 2. 콘텐츠 정보 로드
- 안전 등급 확인 (SAFE/CAUTION/DANGER/FORBIDDEN)
- 음식 정보 로드

### 3. 캡션 작성 (파스타 구조)

```
{안전 이모지} 강아지 {음식명}, {결론 한 줄}

저도 처음에 "{음식명} 먹어도 되나요?" 검색했었어요.
햇살이가 {관련 에피소드}...

{⚠️ 주의사항 (있으면)}
• 항목1
• 항목2

{🚫 금지항목 (있으면)}
• 절대 금지 항목

📏 체중별 급여량
소형견(5kg 이하)은 하루 Xg (체감 단위)
중형견(5~15kg)은 Xg (체감 단위)
대형견(15~30kg)은 Xg (체감 단위)
초대형견(30kg 이상)은 Xg (체감 단위)

{핵심 메시지 - 햇살이 에피소드 포함}

같은 고민을 하고 계실 보호자분들께 도움이 되었으면 좋겠어요!

💡 저장해두고 필요할 때 참고하세요!

📌 이 콘텐츠는 AI의 도움을 받아 제작되었습니다.

#강아지{음식명} #강아지간식 ... (12~16개)
```

### 4. Validator 자동 실행
- caption_validator.py 실행
- 안전 이모지, CTA, AI 고지, 해시태그 검증

### 5. 결과 보고

## 안전 이모지 규칙 (§6.1)

| 등급 | 이모지 | 의미 |
|------|--------|------|
| SAFE | 🟢 | 급여 가능 |
| CAUTION | 🟡 | 주의 필요 |
| DANGER | 🔴 | 위험 |
| FORBIDDEN | 🔴 | 절대 금지 |

## 톤 규칙 (§6.3)

### 필수 표현
- "~더라고요", "~했었어요", "~해보니까" (경험 공유)
- "저도 처음엔 몰랐는데", "검색해봤던 기억이 나요" (동질감)
- 햇살이 이름 직접 언급 + 구체 행동 묘사

### 금지 표현
- "~입니다", "~해주세요"만 반복 (교과서 톤)
- 햇살이 언급 없이 정보만 나열
- "안전합니다", "좋습니다"만 반복 (수의사 톤)

## Validator 체크리스트
- [ ] 안전 이모지 (🟢/🟡/🔴) 존재
- [ ] CTA 표현 (저장/팔로우/공유)
- [ ] AI 고지 포함
- [ ] 해시태그 12~16개
- [ ] 보호자 동질감 메시지
