# /batch blog - 네이버 블로그 배치 처리

네이버 블로그 콘텐츠를 대량 생성합니다.

## 중요: 인스타와 별도 콘텐츠

```
⚠️ 블로그는 인스타와 별도 콘텐츠입니다.
   이미지, 캡션 모두 블로그 전용으로 생성됩니다.
```

| 항목 | 인스타 | 블로그 |
|------|--------|--------|
| 표지 | §8 (영어 제목) | §8 v2.0 (한글 제목) |
| 본문 | 슬라이드 이미지 | HTML 본문 |
| 캡션 | 파스타 구조 | 레드2 확장 모델 |

## 사용 가능한 작업

| 작업 | 설명 | 규칙 |
|------|------|------|
| `cover` | 블로그 표지 생성 | §8 v2.0 |
| `body` | 블로그 본문 HTML | §10 |
| `caption` | 블로그 캡션 | §10 |
| `pipeline` | 표지+본문+캡션 전체 | §8, §10 |
| `publish` | 블로그 발행 준비 | - |
| `validate` | 블로그 Validator | §8, §10 |

## 예시

```bash
/batch blog cover 060-070       # 블로그 표지 11개
/batch blog body 060-070        # 블로그 본문 HTML 11개
/batch blog caption 060-070     # 블로그 캡션 11개
/batch blog pipeline 001-150    # 전체 150개 파이프라인
/batch blog validate 3_approved # 승인 대기 전체 검증
```

## cover 작업

### 인스타 표지와의 차이

| 항목 | 인스타 | 블로그 |
|------|--------|--------|
| 텍스트 언어 | 영어 대문자 | 한글 |
| Y 위치 | 194px | 80px |
| 폰트 크기 | 114px | 120px |
| 그라데이션 | 없음 | 상단 25% |
| 그림자 | 4px | 2px + blur |

### 규칙 (§8 v2.0)
- 폰트: BlackHanSans-Regular 120px
- 드롭쉐도우: offset=2, blur=3, alpha=100
- 상단 그라데이션: 25%, alpha 80→0
- Y 위치: 80px
- **한글 음식명**

### 생성 파일
```
{콘텐츠}/blog/01_표지_{음식명}.png
```

## body 작업

### 실행 내용
1. §10 레드2 확장 모델 적용
2. HTML 본문 생성
3. Cloudinary 이미지 URL 삽입
4. Validator 실행

### 레드2 확장 요소 (§10)

| 요소 | 설명 | 필수 |
|------|------|------|
| 상단 요약 박스 | 결론 먼저 | ✅ |
| 비교표 | 종류별/옵션별 | 해당 시 |
| FAQ 5개 | 검색 질의 대응 | ✅ |
| 간단 레시피 | 조리법 1개 | ✅ |
| 출처 박스 | 전문성 표현 | ✅ |

### 생성 파일
```
{콘텐츠}/blog/blog_body.html
{콘텐츠}/blog/본문.txt
```

## caption 작업

### 블로그 캡션 = 블로그 본문

블로그의 경우 "캡션"이 곧 본문 텍스트입니다.
body 작업과 함께 생성됩니다.

### 추가 생성 내용
- SEO 메타 설명
- 블로그 제목 (§6.5 규칙)
- 태그 목록

### 블로그 제목 규칙 (§6.5)
- 형식: **"강아지 {음식명} 먹어도 되나요?"**
- 질문형 고정
- 금지: 노령견, 급여량, 주의사항, 완벽 정리 등

## pipeline 작업

### 실행 순서
1. cover 실행 → FAIL 시 스킵
2. body 실행 (HTML + 본문) → FAIL 시 스킵
3. 전체 Validator
4. 결과 보고

### 생성되는 파일 전체
```
{콘텐츠}/blog/
├── 01_표지_{음식명}.png
├── 02_음식사진.png
├── 03_영양성분.png
├── 04_급여방법.png
├── 05_급여량.png
├── 06_주의사항.png
├── 07_조리방법.png
├── 08_햇살이.png
├── blog_body.html
├── blog_preview.html
├── 본문.txt
└── README.txt
```

## publish 작업

### 블로그 발행 = 수동

네이버 블로그는 API 자동 게시가 제한적이므로,
publish는 "발행 준비" 상태로 전환합니다.

### 실행 내용
1. 전체 Validator 실행
2. PASS 시 → HTML 최종 생성
3. 복사용 코드 출력
4. "발행 준비 완료" 상태 표시

### 결과
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 블로그 발행 준비 완료: {음식명}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

제목: 강아지 {음식명} 먹어도 되나요?

📎 HTML 파일: blog/blog_body.html
📎 미리보기: blog/blog_preview.html

[복사용 HTML]
(HTML 코드 블록)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## validate 작업

### 검증 항목

```
□ 표지 §8 v2.0
  - 해상도 1080x1080
  - 한글 텍스트
  - Y 위치 80px
  - 그라데이션 존재

□ 본문 §2.1
  - 글자수 1,620~1,980자
  - H2 4개 이상
  - 보호자 동질감 규칙

□ 레드2 확장 §10
  - 상단 요약 박스
  - FAQ 5개
  - 레시피
  - 출처 박스

□ 이미지 구성
  - 8장 존재
  - 8장 햇살이 실사
```

### 결과 형식
```
060: ✅ PASS (전체 통과)
061: ❌ FAIL - 표지 Y위치 120px (80px 필요)
062: ❌ FAIL - FAQ 3개 (5개 필요)
```
