## AUTO_PUBLISH 등급 (v0.4)

### L1: AUTO_PUBLISH

```
대상: SAFE, FORBIDDEN
조건: Agent B 평균 70점+, Agent D Red Flag 0개
처리: 자동 게시 (사람 개입 없음)
```

### L2: HUMAN_QUEUE

```
대상: CONDITIONAL, CAUTION, UNKNOWN
조건: Agent 평가 완료
처리: 사람 확인 후 게시
예시: cheese(저염), egg(익힌것), grape(소량)
```

### L3: HUMAN_QUEUE (강화)

```
대상: DANGER
조건: Agent D 강화 검사 필수
처리: Red Flag 0개 확인 + 사람 승인
```

