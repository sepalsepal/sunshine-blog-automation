## 1. PPL 최소 조건

### 1.1 팔로워 기준

| 등급 | 팔로워 수 | PPL 가능 여부 | 예상 단가 |
|:----:|:---------:|:-------------:|:---------:|
| Nano | 1,000~5,000 | ⚠️ 물물교환 | 제품 제공 |
| Micro | 5,000~10,000 | ✅ 소액 PPL | 10~30만원 |
| Mid | 10,000~50,000 | ✅ 일반 PPL | 30~100만원 |
| Macro | 50,000+ | ✅ 프리미엄 PPL | 100만원+ |

**현재 상태:** Phase 1 (0~1,000) → PPL 영업 불가
**목표:** Phase 2 (3,000+) 도달 시 물물교환 PPL 시작

### 1.1a 권장 기준 vs 최소 허용 기준

| 지표 | 권장 기준 | 최소 허용 기준 | 비고 |
|------|:---------:|:-------------:|------|
| 팔로워 | 5,000+ | 3,000+ | 논의 가능 |
| 참여율 | 12%+ | 8%+ | 업종별 상이 |
| 게시물 수 | 50개+ | 30개+ | 캐러셀 비율 70%+ |
| 운영 기간 | 6개월+ | 3개월+ | 일관된 게시 필수 |

> **참고:** 위 기준은 권장 사항이며, 브랜드/제품 특성에 따라 협의 가능합니다.
> 분기별 시장 상황 반영하여 재평가 예정.

### 1.2 참여율 기준

| 참여율 | 평가 | PPL 협상력 |
|:------:|:----:|:----------:|
| 15%+ | 우수 | 단가 +50% |
| 10~15% | 양호 | 기본 단가 |
| 5~10% | 보통 | 단가 -30% |
| 5% 미만 | 낮음 | PPL 어려움 |

**계산식:**
```
참여율 = (좋아요 + 댓글 + 저장) / 팔로워 × 100
```

### 1.3 콘텐츠 품질 기준

| 항목 | 최소 조건 | 권장 조건 |
|------|:---------:|:---------:|
| 게시물 수 | 30개+ | 50개+ |
| 캐러셀 비율 | 70%+ | 85%+ |
| 브랜드 일관성 | 90%+ | 95%+ |
| 평균 저장 수 | 20+/게시물 | 50+/게시물 |

## 2. 스폰서십 최소 조건

### 2.1 월정액 스폰서십

| 패키지 | 팔로워 조건 | 월 노출 보장 | 월 단가 |
|--------|:----------:|:-----------:|:-------:|
| **Basic** | 5,000+ | 월 4회 제품 노출 | 50만원 |
| **Standard** | 10,000+ | + 스토리 8회 | 100만원 |
| **Premium** | 30,000+ | + 전용 콘텐츠 2개 | 200만원 |

### 2.2 스폰서십 계약 전 필수 조건

```
□ 팔로워 5,000명 이상
□ 3개월 이상 일관된 게시 (주 5회+)
□ 평균 참여율 8% 이상
□ 미디어킷 보유
□ 이전 협업 사례 1건 이상 (물물교환 포함)
```

## 3. 브랜드 컨택 조건

### 3.1 컨택 가능 시점

| Phase | 팔로워 | 컨택 방식 | 기대 결과 |
|:-----:|:------:|-----------|-----------|
| 1 | 0~1,000 | ❌ 컨택 자제 | 브랜딩 집중 |
| 2 | 1,000~3,000 | DM 컨택 | 물물교환 |
| 3 | 3,000~10,000 | 이메일 컨택 | 소액 PPL |
| 4 | 10,000+ | 미디어킷 발송 | 정식 PPL |

### 3.2 미디어킷 필수 항목

```
1. 계정 소개 (1문장)
2. 팔로워 수 / 참여율
3. 주요 타겟층 (반려견 보호자)
4. 대표 콘텐츠 3~5개
5. 이전 협업 사례 (있을 경우)
6. 패키지별 단가표
7. 연락처
```

## 5. 수치 기반 PPL 단가 계산

### 5. 단가 계산 공식

```
기본 단가 = 팔로워 × 0.01 × 참여율 보정
         = 팔로워 × 0.01 × (참여율 / 10)
```

### 예시 계산

| 팔로워 | 참여율 | 계산 | 기본 단가 |
|:------:|:------:|------|:---------:|
| 5,000 | 15% | 5,000 × 0.01 × 1.5 | 75,000원 |
| 10,000 | 12% | 10,000 × 0.01 × 1.2 | 120,000원 |
| 30,000 | 10% | 30,000 × 0.01 × 1.0 | 300,000원 |

### 프리미엄 적용

| 콘텐츠 유형 | 프리미엄 |
|------------|:--------:|
| 피드 1장 | 기본 단가 |
| 캐러셀 (4장) | +50% |
| + 스토리 추가 | +30% |
| + 릴스/영상 | +100% |

## 6. Phase별 영업 로드맵

### Phase 1 (0~3개월): 팔로워 1,000명

```
□ 콘텐츠 품질 확립 (G1/G2/G3 통과율 90%+)
□ 브랜드 일관성 구축
□ 참여율 15% 유지
□ PPL 영업 ❌ (브랜딩 집중)
```

### Phase 2 (3~6개월): 팔로워 3,000명

```
□ 한국 브랜드 물물교환 컨택 시작
□ 타겟: 바잇미, 페슬러, 브리더랩
□ 목표: 물물교환 2~3건
□ 미디어킷 초안 제작
```

### Phase 3 (6~12개월): 팔로워 10,000명

```
□ 정식 PPL 영업 시작
□ 미디어킷 완성 및 발송
□ 타겟: 중고가 브랜드 (PETKIT, HARIO WAN 등)
□ 목표: 유료 PPL 월 1~2건 (50만원/건)
```

### Phase 4 (12개월+): 팔로워 30,000명+

```
□ 프리미엄 브랜드 컨택
□ 월정액 스폰서십 계약
□ 타겟: Le Creuset Pet, Diggs, MiaCara
□ 목표: 월 300만원+ 수익
```

## 7. 체크리스트: PPL 영업 시작 가능?

```
□ 팔로워 3,000명 이상
□ 참여율 10% 이상
□ 게시물 30개 이상
□ 캐러셀 20개 이상
□ 브랜드 일관성 확립 (햇살이 캐릭터)
□ 미디어킷 준비 완료
□ 이전 협업 사례 1건+ (물물교환 포함)

위 조건 5개 이상 충족 시 → PPL 영업 시작 가능
```
