## ⚠️ 리스크 관리 원칙 (필독)

### 1. 수치 제거
- ❌ "소형견 30g, 중형견 50g"
- ✅ "체중 대비 소량", "간식 총량의 10% 이내"

### 2. 단정 표현 회피
- ❌ "급여 가능합니다!"
- ✅ "조건을 지키면 급여 가능합니다"

### 3. 노령견 감량 표현
- ❌ "노령견은 50% 감량"
- ✅ "일반 권장량보다 더 적게"

### 4. 응급 상황 안전망
- ✅ "섭취 여부가 불확실해도 병원 상담 권장"
- ✅ "증상이 없어도 병원 연락 추천"

### 5. 해시태그 제한
- Instagram: 10~15개
- Threads: 3~5개

## ✅ 게시 전 체크리스트

```
□ 분류 확인 (OK/CAUTION/DANGER)
□ 색상 코드 일치
□ "11살 노령견 기준" 문구 포함
□ 수치 제거 확인 (g, 개 등 특정 숫자 없음)
□ 단정 표현 없음 ("절대 안전" 등)
□ AI 표기 포함
□ 해시태그 #KDog노령견 포함
□ 해시태그 개수 준수 (IG 10~15, Threads 3~5)
□ 응급 대처 안전망 문구 (DANGER)
□ Threads 캡션 별도 작성
```
