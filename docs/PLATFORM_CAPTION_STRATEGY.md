## 1. 플랫폼 DNA 분석

### 1.1 Instagram vs Threads 비교

| 요소 | Instagram | Threads |
|------|-----------|---------|
| **톤앤매너** | 정보 전달형, 전문적 | 대화형, 친근함 |
| **길이** | 중간~긴 캡션 (500~1500자) | 짧은 캡션 (100~300자) |
| **구조** | 제목 + 본문 + CTA + 해시태그 | 질문/감정 + 핵심 한줄 + 대화유도 |
| **해시태그** | 필수 (10~20개) | 최소화 (0~3개) |
| **이모지** | 적당히 | 자유롭게 |
| **말투** | ~입니다/~해요 | ~야/~지?/~ㅋㅋ |
| **사용자 기대** | 유용한 정보 습득 | 가벼운 소통, 공감 |

### 1.2 플랫폼별 사용자 심리

```
Instagram 사용자:
"이 음식 강아지한테 줘도 되나? 정확한 정보가 필요해"
→ 신뢰할 수 있는 정보 + 저장해두고 나중에 참고

Threads 사용자:
"아 우리 강아지 이거 좋아하는데 ㅋㅋ 다른 집도 그런가?"
→ 공감 + 댓글로 수다 + 가벼운 정보 공유
```

## 5. 자동 변환 로직

### 5.1 CaptionTransformer 클래스

```python
class CaptionTransformer:
    """Instagram 캡션을 Threads용으로 변환"""

    TONE_MAP = {
        "~입니다": "~야",
        "~합니다": "~해",
        "~하세요": "~해봐",
        "~됩니다": "~돼",
        "확인하세요": "확인해봐",
        "저장하세요": "댓글 ㄱㄱ",
    }

    def transform(self, instagram_caption, food_info):
        # 1. 핵심 정보 추출
        hook = self._create_hook(food_info)
        core_info = self._extract_core(food_info)
        cta = self._create_threads_cta()

        # 2. 톤 변환
        threads_caption = f"{hook}\n{core_info}\n{cta}"

        # 3. 해시태그 제거/최소화
        threads_caption = self._minimize_hashtags(threads_caption)

        return threads_caption
```

### 5.2 변환 파이프라인

```
Instagram 캡션 생성 (김작가)
    ↓
Threads 캡션 자동 변환 (CaptionTransformer)
    ↓
톤앤매너 검수 (김감독)
    ↓
동시 게시 (Instagram + Threads)
```

## 16. AI 표기 정책

> **리뷰어 합의:** 김부장, 우고문, GPT-보수 전원 승인 (2026-01-30)

### 16.1 AI 표기 위치 정책

| 위치 | AI 표기 | 근거 |
|------|:-------:|------|
| **바이오** | ❌ 제외 | 첫인상 공간, 팔로우 전환 우선 |
| **캡션** | ✅ 유지 | 콘텐츠별 투명성 고지 (업계 표준) |

### 16.2 바이오 제외 근거

```
1. 공간 효율성
   - Instagram 바이오: 150자 제한 (현재 ~140자 사용)
   - Threads 바이오: 160자 제한 (현재 ~130자 사용)
   - AI 표기 추가 시 핵심 메시지 공간 부족

2. 역할 분리
   - 바이오 = 첫인상 + 팔로우 유도 (3초 판단)
   - 캡션 = 상세정보 + 투명성 고지

3. 리스크 회피
   - 바이오 AI 명시 → "AI 계정" 프레이밍 위험
   - 캡션 고지 → 읽은 사람만 인지, 불필요한 이탈 방지

4. 업계 표준 준수
   - 정보형/가이드형 계정 대다수 바이오 AI 미표기
   - 개별 게시물 하단 고지가 표준
```

### 16.3 캡션 AI 표기 문구

```
ℹ️ 일부 이미지는 AI로 생성되었습니다.
ℹ️ Some images were generated by AI.
```

> 위치: 캡션 하단 (해시태그 전)
> 참조: CLAUDE.md AI 표기 규칙

### 16.4 플랫폼 정책 변화 대응
