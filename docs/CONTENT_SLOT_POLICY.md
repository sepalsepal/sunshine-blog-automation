## 2. PPL 유형 정의

### 2.1 소프트 PPL (Soft Placement)

| 항목 | 설명 |
|------|------|
| **정의** | 콘텐츠 맥락에 자연스럽게 녹아든 제품 노출 |
| **광고 표기** | 선택 (권장: 표기) |
| **제품 노출** | 배경/소품으로 자연스럽게 |
| **브랜드 언급** | 캡션에서 1회 이하 |
| **예시** | 햇살이 뒤 배경에 브랜드 식기 |

**소프트 PPL 프롬프트 예시:**
```
"... food served in a Le Creuset Pet ceramic bowl ..."
```

### 2.2 하드 PPL (Hard Placement)

| 항목 | 설명 |
|------|------|
| **정의** | 제품이 콘텐츠의 주요 요소로 등장 |
| **광고 표기** | 필수 |
| **제품 노출** | 전면 노출, 로고 가독성 확보 |
| **브랜드 언급** | 캡션에서 2~3회 |
| **예시** | "오리젠 사료와 함께하는 건강한 식사" |

**하드 PPL 광고 표기:**
```
📢 본 콘텐츠는 [브랜드명]의 협찬을 받아 제작되었습니다.
#광고 #협찬 #[브랜드명]
```

## 3. 브랜드 반복 노출 정책

### 3.1 동일 브랜드 노출 간격

| 규칙 | 최소 간격 | 권장 간격 |
|------|:---------:|:---------:|
| 하드 PPL 연속 | 7일 | 14일 |
| 소프트 PPL 연속 | 3일 | 7일 |
| 동일 제품 | 14일 | 30일 |

### 3.2 월간 브랜드 노출 한도

| 브랜드 유형 | 월 최대 | 비고 |
|------------|:-------:|------|
| 단일 브랜드 | 4회 | 하드+소프트 합산 |
| 동일 카테고리 | 6회 | 예: 식기 브랜드 전체 |
| 경쟁 브랜드 | 동시 불가 | 독점 계약 시 |

### 3.3 브랜드 충돌 방지

```
❌ 금지 사례:
- 같은 주에 오리젠 + 아카나 (경쟁 사료)
- 같은 날에 르쿠제펫 + 페슬러 (경쟁 식기)

✅ 허용 사례:
- 오리젠 (사료) + 르쿠제펫 (식기) = 다른 카테고리
- 오리젠 (월초) + 아카나 (월말) = 2주 이상 간격
```

## 4. PPL 스케줄 관리

### 4.1 주간 슬롯 배정

| 요일 | 슬롯 유형 | PPL 가능 | 비고 |
|:----:|----------|:--------:|------|
| 월 | 오가닉 | ❌ | 주 시작, 순수 정보 |
| 화 | 오가닉 | ❌ | |
| 수 | 오가닉 | ❌ | |
| 목 | 오가닉 | ❌ | |
| 금 | 오가닉 | ❌ | |
| **토** | **소프트 PPL** | ⚠️ | 자연스러운 노출 |
| **일** | **하드 PPL** | ✅ | 명시적 협찬 |

### 4.2 PPL 캘린더 예시


## 5. 콘텐츠 믹스 전략

### 5.1 카테고리별 배분 (월간)

| 카테고리 | 비율 | 콘텐츠 수 | PPL 가능 |
|----------|:----:|:---------:|:--------:|
| 과일 | 25% | 7~8개 | ✅ 식기 |
| 채소 | 25% | 7~8개 | ✅ 식기 |
| 육류/단백질 | 20% | 5~6개 | ✅ 사료/간식 |
| 간식/가공식품 | 15% | 4~5개 | ✅ 브랜드 제품 |
| 음료 | 10% | 2~3개 | ⚠️ 제한적 |
| 기타 | 5% | 1~2개 | - |

### 5.2 시즌별 조정

| 시즌 | PPL 비율 조정 | 이유 |
|------|:-------------:|------|
| 1~2월 (신년) | +20% | 브랜드 예산 소진 |
| 5월 (가정의달) | +30% | 반려동물 수요 증가 |
| 11~12월 (연말) | +40% | 홀리데이 시즌 |
| 평시 | 기본 | 5:2 비율 유지 |

## 6. 품질 보호 정책

### 6.1 PPL 거부 사유

| 사유 | 설명 |
|------|------|
| 브랜드 부적합 | 반려견 유해 제품, 저품질 브랜드 |
| 과도한 요구 | 3장 이상 전면 노출, 부자연스러운 연출 |
| 경쟁사 충돌 | 기존 계약 브랜드와 경쟁 |
| 빈도 초과 | 월간 한도 초과 |
| 단가 불일치 | 기준 단가 50% 미만 제안 |

### 6.2 오가닉 콘텐츠 보호

```
핵심 원칙: 오가닉 콘텐츠 품질 > PPL 수익

- 오가닉 슬롯에 PPL 삽입 금지
- PPL로 인한 콘텐츠 품질 저하 시 거부
- 팔로워 피드백 부정적 시 PPL 비율 하향
```

## 7. 측정 및 리포트

### 7.1 PPL 효과 측정

| 지표 | 측정 방법 | 목표 |
|------|----------|:----:|
| PPL 참여율 | PPL 콘텐츠 참여율 / 평균 참여율 | 80%+ |
| 브랜드 언급 | 댓글 내 브랜드 언급 수 | 5+ |
| 링크 클릭 | 프로필 링크 클릭 (추적) | 50+ |
| 팔로워 반응 | 부정 댓글 비율 | 5% 미만 |

### 7.2 월간 PPL 리포트

```
=== 월간 PPL 리포트 ===

1. PPL 현황
   - 총 PPL: X건 (하드 Y, 소프트 Z)
   - 브랜드: A, B, C
   - 총 수익: XXX만원

2. 효과 분석
   - PPL 평균 참여율: X% (전체 대비 Y%)
   - 브랜드 언급: X건
   - 링크 클릭: X회

3. 다음 달 계획
   - 예정 PPL: X건
   - 타겟 브랜드: A, B
```
