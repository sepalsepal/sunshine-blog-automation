# 커버 이미지 규칙

## 버전: 2.0
## 최종 수정: 2026-02-14

---

## 목적

인스타그램/블로그 표지 이미지 제작 시 준수해야 할 디자인 규칙을 정의한다.

---

## 규칙

### 1. 해상도

- **1080 x 1080px 고정** (Instagram 정사각형)
- 1024, 1200 등 다른 해상도 금지

---

### 2. 배경 이미지

- AI 생성 실사풍 이미지
- 구도: 강아지(햇살이) + 음식 접시
- 강아지: 정면 또는 약간 측면, 상반신 이상 노출
- 음식: 하단 1/3 영역에 배치, 그릇에 담긴 형태
- 배경: 실내 (거실/주방), 자연광 느낌

---

### 3. 음식명 텍스트 (한영 통합)

| 항목 | 값 | 비고 |
|------|---|------|
| 내용 | 한글 + 영어 (예: "호박" + "PUMPKIN") | 2줄 구성 |
| 배치 | 한글 위, 영어 아래 | |
| 폰트 | BlackHanSans-Regular | 1순위 (NanumGothic-ExtraBold 대체) |
| 폰트 크기 (한글) | 120px | 글자당 폭 ~114px 기준 |
| 폰트 크기 (영어) | 72px | 한글 대비 60% |
| 영어 대소문자 | **대문자** | ONION, PUMPKIN, APPLE |
| 색상 | #FFFFFF (순백) | RGBA(255,255,255,255) |
| X 정렬 | 중앙 정렬 | center_x = 50% |
| Y 위치 (한글) | 상단 80px | 고정값 |
| Y 위치 (영어) | 상단 210px | 한글 아래 130px |

---

### 4. 드롭 쉐도우 (Drop Shadow)

| 항목 | 값 | 비고 |
|------|---|------|
| 방식 | 별도 레이어 + GaussianBlur | **stroke 사용 금지!** |
| 색상 | RGBA(0,0,0,100) | alpha=100 |
| 오프셋 | X=2px, Y=2px | offset=2 |
| 블러 | radius=3 | ImageFilter.GaussianBlur |

---

### 5. 금지 사항

- 서브타이틀/부제목 금지 (음식명 한영만)
- 상단/하단 그라데이션 금지
- 안전도 배지 금지
- 워터마크 금지
- **stroke_width 사용 금지**
- **Cloudinary 중복 업로드 금지** (기존 URL 존재 시 재업로드 금지)

## 위반 사례

| 위반 | 사례 |
|------|------|
| 해상도 오류 | 1024x1024 사용 |
| stroke 사용 | 드롭 쉐도우 대신 stroke_width 적용 |
| 영문 누락 | 한글만 표시 |
| 영문 소문자 | "pumpkin" (대문자 필수) |
| Y 위치 오류 | 한글 200px 위치 |
| 그라데이션 추가 | 상단/하단 오버레이 |
| 배지 추가 | 안전도 배지 |

---

## Validator 체크리스트

```
□ 해상도 1080x1080
□ 음식명 한글 텍스트 존재
□ 음식명 영어 텍스트 존재 (대문자)
□ 폰트: BlackHanSans-Regular (한글 120px, 영어 72px)
□ 텍스트 색상: 흰색
□ 드롭 쉐도우: offset=2, alpha=100, blur=3
□ Y 위치: 한글 80px, 영어 210px (상단 기준)
□ X 정렬: 중앙
□ 배경 이미지에 강아지 + 음식 포함
□ 그라데이션 없음
□ 배지 없음
```

→ 1개 FAIL = 표지 재제작

## 골든 샘플 비교 (최종 검증)

**필수 단계**: 제작 완료 후, 골든 샘플과 나란히 비교하여 유사성 확인

### 비교 항목

| 항목 | 골든 샘플 기준 | 확인 |
|------|---------------|------|
| 텍스트 위치 | 한글 상단 80px, 영어 210px | □ |
| 텍스트 크기 비율 | 한글 > 영어 (120px vs 72px) | □ |
| 영어 대문자 | ONION, PUMPKIN 등 | □ |
| 드롭쉐도우 | 블러 적용, 오른쪽 아래 방향 | □ |
| 그라데이션 | 없음 | □ |
| 배지 | 없음 | □ |
| 전체 레이아웃 | 강아지+음식 구도 유사 | □ |

### 비교 방법

```
1. 골든 샘플 이미지 열기: ./Golden_Cover_Sample.png
2. 제작 이미지와 나란히 배치 (좌: 골든, 우: 제작)
3. 위 항목별 체크
4. 시각적 유사도 80% 이상 확인
```

### 합격/불합격 기준

```
✅ PASS: 전 항목 체크 + 시각적 유사도 80% 이상
❌ FAIL: 1개 항목 미충족 또는 시각적 유사도 80% 미만
```

→ FAIL 시 STEP 3(생성)으로 롤백
