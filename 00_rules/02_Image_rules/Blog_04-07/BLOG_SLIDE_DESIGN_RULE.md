## 9. 특수문자/이모지 처리 규칙

### 9.1 기본 원칙
```
이모지 사용 금지: 이미지 렌더링 시 이모지(⚠️, ✅, ❌, 🚨, 🔴 등) 직접 사용 금지
이유: 폰트가 이모지를 지원하지 않아 □□ 박스로 표시됨
```

### 9.2 대체 표현 방법

| 원래 이모지 | 대체 방법 | 예시 |
|-------------|-----------|------|
| ⚠️ (경고) | 텍스트 "주의:" 또는 원형 아이콘 | "주의: 첫 급여 시 절반량으로" |
| ✅ (체크) | 초록 원형 아이콘 + "V" | §6.1 참조 |
| ❌ (X표시) | 빨강 원형 아이콘 + "X" | §6.1 참조 |
| 🚨 (경보) | 텍스트 "긴급:" 또는 빨강 배경 박스 | "긴급: 즉시 병원 방문" |
| 💡 (팁) | 텍스트 "TIP:" 또는 노랑 원형 "!" | §7 참조 |

### 9.3 하단 부가 정보 영역 수정

```
경고 메시지:
  - 기존: "⚠️ 첫 급여 시 절반량으로 시작하세요"
  - 수정: "주의: 첫 급여 시 절반량으로 시작하세요"

TIP 메시지:
  - 아이콘: 노랑 원형(24x24) + 흰색 "!" 텍스트
  - 텍스트: "TIP: {내용}"

긴급 메시지:
  - 배경: 빨강 (#FDE8E8)
  - 텍스트: "긴급: {내용}" (빨강 Bold)
```

### 9.4 문서 내 이모지 표기
```
규칙 문서(.md/.txt)에서의 이모지: 가독성 위해 허용
실제 이미지 렌더링 코드: 텍스트 대체어 사용 필수
```
