# Dog with Food 이미지 규칙

## 버전: 1.0
## 최종 수정: 2026-02-14

---

## 목적

음식 준비 장면 이미지 제작 시 준수해야 할 품질 기준을 정의한다.
Cover/Food와 차별화된 "일상 다큐멘터리" 스타일.

---

## 규칙

### 1. 해상도

- **1080 x 1080px 권장** (Instagram 호환)
- 최소 해상도: 1024 x 1024px

---

### 2. 핵심 차별점

| 항목 | Cover/Food | Dog with Food |
|------|------------|---------------|
| 주체 | 햇살이 (강아지) | 음식 준비 장면 |
| 포커스 | 강아지 + 음식 | 음식/손 (전경) |
| 강아지 위치 | 프레임 중앙 | **배경 (보케)** |
| 배경 | 거실 | **주방** |
| 감성 | 초상화 | **일상 다큐멘터리** |

---

### 3. 구도 요소

| 요소 | 기준 | 포커스 |
|------|------|--------|
| 음식/손 | 전경 중앙~하단 | **Sharp** |
| 강아지 | 배경 상단 | **Soft (보케)** |
| 주방 | 컨텍스트 제공 | Medium |

---

### 4. 전경 (Sharp Focus)

```
필수 요소:
- 사람 손 (얼굴은 프레임 밖)
- 음식 준비 동작 (자르기, 담기, 계량 등)
- 음식 명확히 식별 가능
- 적절한 조리도구/그릇
```

**허용 동작:**
- 나무 숟가락으로 음식 담기
- 주방 저울에 올리기
- 도마에서 자르기
- 유리볼에 섞기
- 계량컵으로 덜기

---

### 5. 배경 (Soft Focus)

```
강아지 위치:
- 주방 바닥에 앉아서 지켜봄
- 카운터 뒤에서 냄새 맡음
- 머리 기울여 호기심 표현
- 근처에 누워 눈으로 따라감
- 카운터 모서리 뒤에서 엿봄
```

**강아지 특징:**
- 11살 시니어 골든 리트리버 "햇살이"
- 흰 주둥이 (노령견 특징)
- 카메라 응시 금지 (음식/손 쪽 시선)
- 자연스럽고 비연출적

---

### 6. 주방 설정

```
- 2026년 한국 아파트 주방
- 따뜻한 톤, 모던 미니멀
- 스톤 조리대, 간접 조명
```

**조리대 옵션:** 화이트 마블 | 천연 그래닛 | 쿼츠 | 우드 도마
**캐비닛 옵션:** 웜 우드톤 | 매트 화이트 | 세이지 그린 | 크림

---

### 7. 조명

```
- 위에서 따뜻한 주방 조명
- 측면 창에서 자연광
- 부드럽고 따뜻한 분위기
- 강한 그림자 금지
```

---

### 8. 금지 사항

```
❌ 강아지가 주인공처럼 중앙 배치
❌ 강아지에 샤프 포커스
❌ 사람 얼굴 노출
❌ 거실 배경 (주방만 허용)
❌ 연출된 느낌 (자연스러운 일상 필수)
❌ 프롬프트 임의 수정
```

참조: `./Golden_DogWithFood_Sample.png`

## Validator 체크리스트

```
□ 해상도 1080x1080 이상
□ 전경: 음식+손 (샤프 포커스)
□ 배경: 강아지 (소프트 보케)
□ 강아지 카메라 미응시
□ 사람 얼굴 미노출
□ 주방 배경
□ 음식 명확히 식별 가능
□ 일상적/자연스러운 분위기
```

→ 1개 FAIL = 이미지 재생성

---

## 골든 샘플 비교 (최종 검증)

**필수 단계**: 제작 완료 후, 골든 샘플과 나란히 비교하여 유사성 확인

### 비교 항목

| 항목 | 골든 샘플 기준 | 확인 |
|------|---------------|------|
| 포커스 분리 | 전경 샤프, 배경 보케 | □ |
| 음식/손 위치 | 중앙~하단, 명확 | □ |
| 강아지 위치 | 배경, 소프트 포커스 | □ |
| 강아지 시선 | 음식/손 방향 (카메라 X) | □ |
| 배경 | 주방 (거실 아님) | □ |
| 분위기 | 일상 다큐멘터리 | □ |

### 비교 방법

```
1. 골든 샘플 이미지 열기: ./Golden_DogWithFood_Sample.png
2. 제작 이미지와 나란히 배치 (좌: 골든, 우: 제작)
3. 위 항목별 체크
4. 시각적 유사도 80% 이상 확인
```

### 합격/불합격 기준

```
✅ PASS: 전 항목 체크 + 시각적 유사도 80% 이상
❌ FAIL: 1개 항목 미충족 또는 시각적 유사도 80% 미만
```

→ FAIL 시 이미지 재생성
