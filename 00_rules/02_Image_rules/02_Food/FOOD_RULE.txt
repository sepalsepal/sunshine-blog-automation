# 음식 이미지 규칙

## 버전: 1.0
## 최종 수정: 2026-02-14

---

## 목적

블로그 Common_02_Food 이미지 제작 시 준수해야 할 품질 기준을 정의한다.

---

## 규칙

### 1. 해상도

- **1080 x 1080px 권장** (Instagram 호환)
- 최소 해상도: 1024 x 1024px

---

### 2. 구도 요소

| 요소 | 기준 | 비고 |
|------|------|------|
| 강아지 | 햇살이 (시니어 골든 리트리버) | 흰 주둥이 특징 |
| 음식 | 그릇에 담긴 상태 | 브랜드 제품은 원본 포장 |
| 배경 | 한국 아파트 거실 | 자연광 느낌 |
| 배치 | 강아지 중앙, 음식 하단 | 음식 강조 |

---

### 3. 강아지 특징 (필수)

```
✅ 필수:
- 11살 시니어 골든 리트리버 암컷 "햇살이"
- 흰 주둥이와 얼굴 (노령견 특징)
- 골든 털, 검은 코와 눈
- 자연스러운 포즈 (경직되지 않음)

❌ 금지:
- 젊은 강아지 묘사
- 다른 견종
- 과도하게 연출된 포즈
```

---

### 4. 음식 표현

| 유형 | 담기 방식 |
|------|----------|
| 일반 음식 (사과, 당근 등) | 펫 브랜드 그릇 |
| 브랜드 제품 | 원본 포장 그대로 |
| 음료 | 오리지널 컵/병 |

---

### 5. 배경 요소

```
- 2026년 한국 아파트 거실
- 간접 조명 (따뜻한 톤)
- 3날개 실링 팬
- 관엽식물 (몬스테라, 아레카야자 등)
- Mr. Maria Brown 램프 (창가)
```

---

### 6. 금지 사항

```
❌ 프롬프트 임의 수정 (음식명만 교체 가능)
❌ 인위적/부자연스러운 조명
❌ 음식 과장/왜곡
❌ 강아지 얼굴 가림
❌ 배경 과도한 노출
```

## 골든 샘플

참조: `./Golden_Food_Sample.png`

---

## Validator 체크리스트

```
□ 해상도 1080x1080 이상
□ 햇살이 특징 (흰 주둥이, 골든 털)
□ 음식 명확하게 식별 가능
□ 적절한 담기 방식 (그릇/포장)
□ 한국 아파트 거실 배경
□ 자연스러운 포즈
□ 따뜻한 조명
□ 음식과 강아지 비율 적절
```

→ 1개 FAIL = 이미지 재생성

---

## 골든 샘플 비교 (최종 검증)

**필수 단계**: 제작 완료 후, 골든 샘플과 나란히 비교하여 유사성 확인

### 비교 항목

| 항목 | 골든 샘플 기준 | 확인 |
|------|---------------|------|
| 강아지 특징 | 흰 주둥이 시니어 골든 리트리버 | □ |
| 강아지 위치 | 프레임 중앙~상단 | □ |
| 음식 위치 | 하단, 그릇에 담김 | □ |
| 배경 분위기 | 따뜻한 거실, 보케 처리 | □ |
| 조명 | 자연광/따뜻한 간접 조명 | □ |
| 전체 품질 | 고품질 실사풍 | □ |

### 비교 방법

```
1. 골든 샘플 이미지 열기: ./Golden_Food_Sample.png
2. 제작 이미지와 나란히 배치 (좌: 골든, 우: 제작)
3. 위 항목별 체크
4. 시각적 유사도 80% 이상 확인
```

### 합격/불합격 기준

```
✅ PASS: 전 항목 체크 + 시각적 유사도 80% 이상
❌ FAIL: 1개 항목 미충족 또는 시각적 유사도 80% 미만
```

→ FAIL 시 이미지 재생성
