# 햇살이 실사 이미지 규칙

## 버전: 1.1
## 최종 수정: 2026-02-14

---

## 목적

블로그/인스타 8장 마무리 슬라이드에 사용할 햇살이 실사 이미지 규칙을 정의한다.

---

## 규칙

### 1. 이미지 사양

| 항목 | 값 |
|------|---|
| 해상도 | 1080x1080 |
| 용도 | 8장 마무리 이미지 (햇살이 실사) |

---

### 2. 이미지 유형

**⚠️ 8장(햇살이 실사) 규칙:**
- 반드시 **실사 햇살이 사진** 사용
- AI 생성 이미지 ❌ 금지
- C2 인포그래픽 ❌ 금지
- 텍스트 오버레이 없는 순수 실사 사진

---

### 3. 허용되는 이미지

- 햇살이 정면/측면 사진
- 햇살이 + 음식 사진
- 햇살이 일상 사진
- 고품질 실사

---

### 4. 금지 사항

| 금지 | 이유 |
|------|------|
| AI 생성 이미지 | 마무리는 실사 원칙 |
| 텍스트 오버레이 | 순수 사진 원칙 |
| 스톡 이미지 | 햇살이 사진만 허용 |
| 다른 강아지 사진 | 햇살이만 허용 |
