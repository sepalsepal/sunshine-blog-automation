# 캡션 마스터 규칙

## 버전: 1.0
## 최종 수정: 2026-02-14

---

## 목적

모든 플랫폼(인스타그램, 쓰레드, 블로그) 캡션 작성 시 준수해야 할 공통 규칙을 정의한다.

---

## 규칙

### 1. 제목 규칙 (필수)

**형식:** `"강아지 {음식명} 먹어도 되나요?"`

- 질문형 고정
- 다른 형식 금지

**금지 표현:**
- 노령견, 급여량, 주의사항, 완벽 정리 등 한정/나열 표현

**예시:**
```
✅ 강아지 호박 먹어도 되나요?
✅ 강아지 연어 먹어도 되나요?
❌ 11살 노령견 호박 급여량·주의사항
❌ 호박 급여량 가이드
```

---

### 2. 캡션 구조 (파스타 규칙)

모든 캡션은 아래 순서를 따른다:

1. **결론** (안전 이모지 + 안전도 답변)
2. **주의사항** (주의 리스트)
3. **금지사항** (해당 시)
4. **급여량** (체중별 4단계)
5. **핵심 메시지**
6. **CTA** (Call to Action)
7. **AI 고지**
8. **해시태그** (인스타/블로그만, 쓰레드 제외)

---

### 3. 급여량 4단계 (필수)

| 구분 | 체중 |
|------|------|
| 소형견 | 5kg 이하 |
| 중형견 | 5~15kg |
| 대형견 | 15~30kg |
| 초대형견 | 30kg 이상 |

**형식:**
```
소형견(5kg 이하)은 하루 10~20g (한두 조각)
중형견(5~15kg)은 20~40g (어른 손 한 줌 정도)
대형견(15~30kg)은 40~60g (종이컵 반 컵 정도)
초대형견(30kg 이상)은 60~80g이 적당합니다 (종이컵 한 컵)
```

**필수:**
- 4줄 고정 (뭉쳐 쓰기 금지)
- 그람 수치 + 체감 단위 필수

**체감 단위 가이드:**

| 그람 범위 | 체감 단위 예시 |
|-----------|----------------|
| 5~10g | 엄지손톱 크기, 한 조각 |
| 10~20g | 한두 조각, 티스푼 2~3개 |
| 20~40g | 어른 손 한 줌, 탁구공 크기 |
| 40~60g | 종이컵 반 컵, 주먹 반 개 |
| 60~80g | 종이컵 한 컵, 어른 주먹 하나 |
| 80~100g | 밥공기 반 그릇 |

---

### 4. 안전도별 톤

| 안전도 | 톤 | 핵심 표현 |
|--------|-----|----------|
| 🟢 SAFE | 밝고 가벼움 | 맘껏, 추천, 좋아해요 |
| 🟡 CAUTION | 친절한 주의 | 조건부 OK, 근데, 주의 |
| 🔴 DANGER | 진지한 경고 | 위험, 주의, 소량만 |
| ⛔ FORBIDDEN | 강한 경고 | 금지, 절대, 응급 |

---

### 5. AI 고지 (필수)

모든 캡션 끝에 포함:
```
ℹ️ 일부 이미지는 AI로 생성되었습니다.
```

또는:
```
ℹ️ 일부 이미지는 AI로 생성되었습니다
```

---

### 6. 해시태그 규칙

**인스타/블로그:** 12~16개
**쓰레드:** 없음 (해시태그 금지)

**필수 해시태그:**
- #강아지{음식명}
- #강아지간식
- #반려견음식

---

### 7. 보호자 동질감 규칙

**원칙:** "수의사 블로그"가 아니라 "같은 견주 블로그"

| # | 위치 | 필수 요소 |
|---|------|----------|
| 1 | 도입부 | "나도 이걸 검색했었다" 경험 고백 |
| 2 | 본문 중간 | 햇살이 실제 경험 에피소드 최소 2회 |
| 3 | 마무리 | "같은 고민 하는 보호자에게" 메시지 |

**톤 규칙:**
```
✅ "~더라고요", "~했었어요", "~해보니까"  (경험 공유)
✅ "저도 처음엔 몰랐는데", "검색해봤던 기억이 나요"  (동질감)
✅ 햇살이 이름 직접 언급 + 구체 행동 묘사  (실감)
❌ "~입니다", "~해주세요"만 반복  (교과서 톤)
❌ 햇살이 언급 없이 정보만 나열  (동질감 부재)
❌ "안전합니다", "좋습니다"만 반복  (수의사 톤)
```

---

### 8. FORBIDDEN 안전도 특수 규칙

FORBIDDEN 음식 캡션에는:
- ❌ 급여법 포함 금지
- ❌ 조리법 포함 금지
- ✅ 위험 이유 필수
- ✅ 응급 대응 안내 필수

## 위반 사례

| 위반 | 사례 |
|------|------|
| 제목 형식 | "호박 급여 가이드" (질문형 아님) |
| 급여량 누락 | 소형견/대형견 2단계만 표시 |
| 체감 단위 누락 | "20~40g" (비유 없음) |
| 해시태그 과다 | 20개 이상 |
| AI 고지 누락 | 문구 없음 |
| FORBIDDEN 위반 | 포도 캡션에 조리법 포함 |

## 체크리스트 (자가 검증)

```
□ 제목: "강아지 {음식명} 먹어도 되나요?" 형식
□ 구조: 결론 → 주의 → 금지 → 급여 → 핵심 → CTA → AI고지 → 해시태그
□ 급여량: 4단계 모두 포함
□ 체감 단위: 그람 + 비유 표현 모두 있음
□ 안전도 톤: food_data와 일치
□ 햇살이 언급: 경험담 포함
□ CTA: 끝부분에 존재
□ AI 고지: 문구 존재
□ 해시태그: 12~16개 (인스타/블로그)
□ FORBIDDEN: 급여법/조리법 없음
```
