## 1. 공통 규칙

### 1.1 필수 요소 (모든 플랫폼)

| 요소 | 필수 | 설명 |
|------|------|------|
| 제목 | ✅ | 질문형 ("~먹어도 되나요?") |
| 햇살이 언급 | ✅ | "우리 햇살이" 포함 |
| CTA | ✅ | 질문/공유 유도 |
| 안전도 톤 | ✅ | SAFETY_MATRIX.md 준수 |

### 1.2 제목 형식

```
허용 패턴:
✅ "강아지 {음식} 먹어도 되나요?"
✅ "{음식} 강아지한테 줘도 되나요?"
✅ "{음식} 강아지한테 줘도 될까요?"
✅ "{음식}, 강아지가 먹어도 될까요?"

금지:
❌ "강아지 {음식} 완벽 정리"
❌ "{음식} 급여량과 주의사항"
❌ 서술형 제목
```

### 1.3 화자 설정

```
화자: 11살 골든리트리버 햇살이 엄마
톤: 따뜻한 경험 공유, 동료 보호자 느낌

권장 표현:
✅ "안녕하세요, 11살 골든리트리버 햇살이 엄마예요"
✅ "우리 햇살이한테 줘봤더니..."
✅ "저희 아이는..."

금지 표현:
❌ "햇살이네 강아지 음식 연구소"
❌ "전문가에 따르면"
❌ "연구 결과"
```

## 2. 인스타그램 규칙

### 2.1 구조 (SAFE/CAUTION)

```

{결론 뱃지}

🟡 {음식} 급여 시 주의사항
• {주의1}
• {주의2}
• {주의3}

❌ 절대 금지 항목
• {금지1} ({이유})
• {금지2} ({이유})

📏 급여 방법
• 소형견: {수량}
• 중형견: {수량}
• 대형견: {수량}
※ {조리법}
※ 가끔 간식으로만 OK

📌 기억하세요
"{핵심 한줄}"

💾 저장해두고 주변에 공유하세요!
건강한 간식 정보, 함께 나눠요 🐶

{해시태그 12~16개}
```

### 2.2 구조 (FORBIDDEN)

```
{이모지} {음식}, 강아지가 먹어도 될까요?

⛔ 결론: 절대 금지!

🔴 {음식} 위험 이유
{독성 설명}

🚨 응급 상황 대처
• {응급 조치}
• 구토 유도 시도하지 마세요
• 증상: {증상 목록}

📌 기억하세요
"{음식} = 절대 금지"
모르고 주시는 분들 많아서 공유해요 🐕

{해시태그}
```

### 2.3 해시태그 규칙

```
개수: 12~16개 (권장 16개)

필수 해시태그 (10개):
#강아지{음식} #강아지간식 #반려견음식 #강아지건강
#펫푸드 #강아지케어 #골든리트리버 #시니어독
#강아지정보 #반려견가이드

선택 해시태그 (2~6개):
#강아지음식가이드 #dogfood #doghealth #petcare
#goldensofinstagram #노령견간식
```

### 2.4 결론 뱃지

```
SAFE: ✅ 결론: 안전합니다!
CAUTION: ⚠️ 결론: 주의가 필요합니다!
DANGER: 🔴 결론: 위험합니다!
FORBIDDEN: ⛔ 결론: 절대 금지!
```

## 3. 쓰레드 규칙

### 3.1 핵심 특성

| 항목 | 인스타그램 | 쓰레드 |
|------|-----------|--------|
| 톤 | 파스타 스타일 | 대화체 (친구처럼) |
| 길이 | 길게 | 짧게 (5-7줄) |
| 해시태그 | 12-16개 | **없음** |
| 이모지 | 절제 | 자연스럽게 (3개 이하) |

### 3.2 구조 (SAFE)

```
{음식} 강아지한테 줘도 되나요? {이모지}
우리 햇살이 {음식} 완전 좋아해요!

{음식}은 강아지한테 정말 좋은 간식이에요.
{효능1}도 있고 {효능2}에도 좋아요.
간식으로 맘껏 줘도 돼요~ 🐕

여러분 강아지도 {음식} 좋아하나요?

```

### 3.3 구조 (CAUTION)

```
{음식} 강아지한테 줘도 될까요? {이모지}
우리 햇살이는 {조리법} {음식} 좋아해요!

{효능} 좋지만, 조건이 있어요!
✔️ {조건1} ✔️ {조건2} ❌ {금지사항}

여러분 강아지는 {음식} 좋아하나요? 🐕

```

### 3.4 구조 (FORBIDDEN)

```
{음식} 강아지한테 절대 주면 안 돼요! {이모지}
우리 햇살이도 절대 안 줘요!

{독성 이유}
소량만 먹어도 위험해요.
🚨 먹었다면 → 즉시 동물병원! (먹은 양/시간 기억)

모르고 주시는 분들 많아서 공유해요.

```

### 3.5 금지 사항

```
❌ 해시태그 사용
❌ "ㅋㅋ", "ㅎㅎ" 자음
❌ 이모지 4개 이상
❌ 10줄 초과
❌ 전문용어 남발
```

## 5. 급여량 4단계 (FORBIDDEN 제외)
