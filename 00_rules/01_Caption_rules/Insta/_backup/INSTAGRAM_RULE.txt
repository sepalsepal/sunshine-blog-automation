# 인스타그램 캡션 규칙

## 버전: 1.0
## 최종 수정: 2026-02-14

---

## 목적

인스타그램 플랫폼 전용 캡션 작성 규칙을 정의한다.

---

## 규칙

### 1. 기본 설정

| 항목 | 값 |
|------|-----|
| 해상도 | 1080x1080 |
| 폰트 | NotoSansCJK |
| Validator | 15개 통과 필수 |
| 캡션 스타일 | 파스타 규칙 |

---

### 2. 캡션 템플릿 v1.0

```
🥕 강아지 [음식명], 줘도 되나요?

[안전도에 따른 답변]

📏 체중별 급여량

소형견 (5kg 이하) — [g범위] ([직관적 단위])
중형견 (5~15kg) — [g범위] ([직관적 단위])
대형견 (15~30kg) — [g범위] ([직관적 단위])
초대형견 (30kg 이상) — [g범위] ([직관적 단위])

✅ 급여 팁
• [팁1]
• [팁2]
• [팁3]

[햇살이 경험담 1~2문장]

처음 주실 땐 조금만! 반응 보고 늘려주세요.

#강아지[음식명] #강아지간식 #반려견음식 ...
```

---

### 3. 필수 요소

- 4단계 급여량 (소형/중형/대형/초대형)
- 직관적 단위 (조각, 개, 스푼 등)
- 햇살이 경험담
- 해시태그 12~16개

---

### 4. 금지 사항

- 임의 요약/축약
- 급여량 단계 생략
- 직관적 단위 생략
- 해시태그 12개 미만 또는 17개 이상

## Validator 체크리스트

```
□ 제목: 질문형
□ 안전도 이모지 포함
□ 급여량 4단계 모두 포함
□ 직관적 단위 포함
□ 햇살이 언급 포함
□ 급여 팁 3개 이상
□ CTA 문구 포함

□ 해시태그 12~16개
□ 안전도와 톤 일치
```
