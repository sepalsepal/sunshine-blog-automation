## 규칙 우선순위

```

```

## 공장형 전환 핵심 문서

### 안전도 정의 (SSOT)
`01_Caption/SAFETY_MATRIX.md`
- SAFE / CAUTION / DANGER / FORBIDDEN 정의
- FORBIDDEN 하드블록 규칙
- 색상 코드
- Validator 안전도 검증 기준

### 캡션 규칙 (통합)
`01_Caption/CAPTION_RULE.md`
- 3플랫폼 공통 규칙
- 플랫폼별 섹션 (인스타/쓰레드/블로그)
- Validator 체크리스트

### 템플릿 파일
`01_Caption/{Platform}/{Safety}/Golden_{Platform}_{Safety}_Template.txt`
- 12개 템플릿 (3플랫폼 × 4안전도)
- 변수 목록 + 체크리스트 포함
- FORBIDDEN 템플릿은 급여량/조리법 섹션 하드 제거

## 빠른 참조

### 캡션 생성 시 (공장형)
1. `01_Caption/SAFETY_MATRIX.md` 안전도 확인
2. `01_Caption/CAPTION_RULE.md` 규칙 확인
3. `01_Caption/{Platform}/{Safety}/Golden_{Platform}_{Safety}_Template.txt` 템플릿 사용
4. Validator 3단계 통과 확인

### 이미지 생성 시
1. `02_Image/COVER_RULE.md` 또는 `INFOGRAPHIC_RULE.md` 확인
2. 해당 폴더 골든 샘플 참조 (01_Cover/, 02_Food/, 03_DogWithFood/, Blog_04-07/{Safety}/ 등)
3. Validator 체크리스트 확인

### QC 수행 시
1. `04_QC/QC_CHECKLIST.md` 전체 읽기
2. 불합격 사례 `REJECTION_CASES.md` 확인
